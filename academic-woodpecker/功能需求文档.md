## 1. 引言

### 1.1 项目背景

随着教育信息化的深入发展，学生面临着越来越繁重的学业任务和多样化的学习资源。如何高效地诊断学习薄弱点、针对性地复习提升，成为学生和教育工作者共同关注的问题。

**学业啄木鸟**应运而生，它是一款基于人工智能技术的智能学习分析平台，旨在帮助学生：
- 🔍 **精准诊断**：通过分析成绩单、笔记等文档，识别知识薄弱点
- 📊 **可视化呈现**：以知识图谱形式直观展示学习状态
- 🎯 **个性化推荐**：AI智能推荐符合学生需求的学习资源
- 💬 **随时答疑**：AI助手7×24小时提供学习辅导

### 1.2 项目目标

本项目的核心目标是构建一个**智能化、个性化、可视化**的学习分析平台，具体包括：

| 序号 | 目标描述 | 优先级 |
|------|---------|--------|
| M1 | 实现学情数据的自动化采集与分析 | P0 |
| M2 | 提供直观易懂的知识掌握可视化展示 | P0 |
| M3 | 基于AI的个性化资源推荐 | P1 |
| M4 | 实现智能问答辅导功能 | P1 |
| M5 | 支持学习计划的制定与跟踪 | P2 |

### 1.3 术语定义

| 术语 | 定义 |
|------|------|
| 学情数据 | 学生的学习行为、成绩、作业等数据的统称 |
| 知识图谱 | 以节点和连线形式展示知识体系及其关联关系的可视化图 |
| 薄弱知识点 | 学生掌握程度不达标、需要加强学习的知识点 |
| OCR识别 | 光学字符识别技术，用于从图片中提取文字 |
| 上下文感知 | AI根据对话历史和学生背景信息提供个性化回复的能力 |

## 3. 功能需求详细说明

### 3.1 模块一：个人学情仪表盘（Dashboard）

#### 3.1.1 功能概述

个人学情仪表盘是用户登录后的主页面，承担着**现状呈现**的核心职责。页面以直观的可视化方式展示用户的整体学习状态，让用户在第一时间了解自己的学习情况。

#### 3.1.2 功能需求清单

| 功能编号 | 功能名称 | 功能描述 | 优先级 | 状态 |
|---------|---------|---------|--------|------|
| F-D-01 | 学情概览卡片 | 展示知识掌握率、已分析文档数、最近诊断时间等核心指标 | P0 | ✅ |
| F-D-02 | 动态知识图谱 | 以放射状图谱展示知识点及其掌握状态，支持点击交互 | P0 | ✅ |
| F-D-03 | 薄弱点详情 | 点击薄弱节点显示详细分析和推荐资源 | P0 | ✅ |
| F-D-04 | 快捷入口 | 提供"新文档分析"和"查看今日推送"快捷操作 | P1 | ✅ |
| F-D-05 | 实时数据更新 | 知识图谱和指标数据实时更新 | P1 | 待开发 |

| 异常场景 | 处理方式 |

| 首次登录无数据 | 显示引导页，引导用户上传第一个文档 |
| 数据加载失败 | 显示重试按钮，提供错误提示 |
| 知识点数据为空 | 显示占位图，提示"暂无数据" |
| 网络连接中断 | 缓存展示旧数据，显示离线状态标识 |

### 3.3 模块三：个性化推送与资源库（Resource Hub）

#### 3.3.1 功能概述

资源库模块利用AI的联网能力，从全网聚合优质学习资源，并根据用户的学情进行个性化推荐，真正实现"因材施教"。

#### 3.3.2 功能需求清单

| 功能编号 | 功能名称 | 功能描述 | 优先级 | 状态 |
|---------|---------|---------|--------|------|
| F-R-01 | 资源瀑布流 | 展示AI推荐的优质学习资源 | P0 | ✅ |
| F-R-02 | AI资源评级 | 每个资源标注推荐理由 | P0 | ✅ |
